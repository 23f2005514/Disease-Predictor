# Disease-Predictor
This project is a simple machine learning-based disease prediction system built using lightweight models (SML) and deployed with a Streamlit web interface. It allows users to input basic symptoms and get a predicted disease as output.

🚀 Features
🧠 Uses a Decision Tree Classifier for prediction
📊 Accepts symptom-based inputs (fever, cough, headache, etc.)
⚡ Fast and lightweight (suitable for deployment on platforms like Hugging Face Spaces)
🖥️ Interactive UI built with Streamlit
🔍 Easy to extend with more symptoms and diseases
